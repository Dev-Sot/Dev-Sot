## About Me

I am a **Systems Engineering student** and **Full Stack Developer** with a strong backend foundation and growing frontend skills. I build systems that are efficient, scalable, and maintainable — not just code that works, but code that *lasts*.

- Backend is my foundation: **Java + Spring Boot**, **Python + FastAPI / Flask**
- Frontend in progress: **React + TypeScript + TailwindCSS**
- I work daily with **Linux**, Bash, Docker, and Nginx
- Strong focus on **software architecture** and **API design**
- Open to **freelance projects**, collaborations, and new challenges
- Currently consolidating my full stack with **Next.js** and cloud deployment
